# lunch_hackathon

## Server
- brew install python (install python 2.7)
- pip install virtualenv
- virtualenv venv
- . venv/bin/activate
- deactivate
- pip install Flask
- pip install Flask-OAuth

## UI

To build and run:

- `npm install -g gulp`
- `npm install`
- `gulp build`
- `python -m SimpleHTTPServer 1337`
- visit `http://localhost:1337`
